Description

This project is a web application built with React.js. It provides functionality for users to register, log in, and interact with various features ( under the task manager option). 

Technologies Used
Frontend: React.js, HTML5, CSS3, JavaScript

Database: MySQL

Tools: Git, Visual Studio Code, npm


Instructions to Run Project

Clone the repository:
git clone https://github.com/keitumetsemahempe-hub/web-design-and-dev---INFS-202.git

Navigate into the project folder:
cd frontend
Install dependencies:
npm install

Start the project:
npm start

This will run the app on https://web-design-and-dev-infs-202-8z6x.vercel.app/
