React Midterm Project
Project Description

A web application built with React.js for managing tasks and projects. The app allows users to create, view, and track their tasks easily.

Tech Stack
Frontend: React.js, Vite, CSS
Backend: Node.js, Express.js (if applicable)
Version Control: Git, GitHub
Deployment: Vercel
Setup Instructions

Clone the repository:

git clone https://github.com/keitumetsemahempe-hub/web-design-and-dev---INFS-202.git

Install dependencies:

cd frontend
npm install

Run the development server:

npm run dev

Run the backend :

cd backend
npm install
npm run dev
